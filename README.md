# Low-level programming lab1
### Вариант 1: Документное дерево
#### Сборка
+ Linux:
```
git clone https://github.com/Gramdel/llp_lab1
cd llp_lab1
make all
./demo
./run_tests
```
+ Windows:
```
git clone https://github.com/Gramdel/llp_lab1
cd llp_lab1
make -f Makefile.win all
.\demo.exe
.\run_tests.exe
```
